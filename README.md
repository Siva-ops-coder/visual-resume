# 🌐 Visual Resume Builder

A modern, responsive, and interactive **Visual Resume** built using HTML, CSS, and JavaScript. This project lets you showcase your skills, experience, and personal brand in a visually engaging way, and it's perfect for deploying as a GitHub Pages portfolio.


## 📌 Features

- 🧑‍💼 Profile Section with picture and contact info
- 🛠️ Skills section with progress bars or badges
- 🎓 Education and 📈 Experience timelines
- 📂 Projects with live/demo links
- 📄 Downloadable PDF resume button
- 💡 Clean UI using modern CSS layout (Flexbox/Grid)
- 🌐 Hosted using GitHub Pages
